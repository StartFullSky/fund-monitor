# 🤖 基金持仓异动监控

基于 Hermes Agent 的智能基金监控系统，按板块汇总推送 Bark 异动通知。

## 功能

- **基金持仓异动监控** — 5大板块、30+只核心股票实时监控
- **金价监控** — 浙商积存金趋势状态机 + Bark 推送
- **盘前晨报** — 每日开盘前推送大盘+关注股票+涨停统计
- **技术分析** — K线 MA/RSI/MACD 分析
- **财经日历** — 重要经济事件推送

## 数据源

- A股行情: 腾讯 `qt.gtimg.cn`
- K线数据: 新浪 `quotes.sina.cn`
- 金价: 金十数据 MCP
- 财经日历: 东方财富
- 推送: Bark (iOS)
